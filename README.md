# line_present

LINE感想プレゼント企画ページです。

## Published pages

- 終了固定ページ: https://minori-kosaka.github.io/line_present/
- 期間連動ページ: https://minori-kosaka.github.io/line_present/event.html
- 既存URL互換: https://minori-kosaka.github.io/line_present/normal.html

## Behavior

`event.html` と `normal.html` は同じキャンペーンページです。

- 2026年6月3日 23:59:59 までは応募受付中の見た目を表示します。
- 2026年6月4日 0:00 以降は受付終了の見た目に自動で切り替わります。
- `?state=closed` を付けると、締切前でも終了状態を確認できます。

`index.html` は常に受付終了の固定ページとして公開します。
